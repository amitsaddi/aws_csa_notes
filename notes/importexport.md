# Import/Export 

Migration tools for large data (to and from AWS)

Two modes
## Snowball
    - Encrypted by default
    - Physical device provided by AWS
    - Import/Export to S3
    - Large volue (to 80TB)
## Disk
    - You provide the disc
    - You can import to Glacier, EBS and S3
    - Export from S3
    - Encryption optional
    - Up to 16TB
